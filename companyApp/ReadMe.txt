Create simple register of companies. Registration should be written in PHP and executed from CLI (https://www.php.net/manual/en/features.commandline.usage.php).
Please add your code to www.github.com or www.bitbucket.org and share the link of your repository.
Application should have the following fields:
	* 
id
	* 
name
	* 
registration_code
	* 
email
	* 
phone
	* 
comment


Application should have the following functionality:
	* 
Add company
	* 
Edit company
	* 
Delete company


Bonus points if you:
	* 
Don’t use any framework
	* 
Do validations (unique email, valid phone number)
	* 
Add ability to import companies in .csv format. Example of .csv file should be added to repository
